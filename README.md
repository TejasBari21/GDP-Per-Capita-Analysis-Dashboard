# GDP Per Capita Analysis Dashboard

## Project Overview

This project is an interactive data analytics dashboard developed using Python and Plotly to visualize and analyze global economic and demographic indicators.

The dashboard provides insights into GDP per Capita, Population, and Life Expectancy across different countries and continents through interactive visualizations and dynamic filters, enabling users to explore global development trends and compare regional performance.

## Features

* Interactive dashboard built with Plotly
* Multi-select filters for Countries and Continents
* Year-wise data exploration
* GDP per Capita analysis across countries
* Population comparison by country
* Life Expectancy vs GDP visualization
* Continent-wise economic performance analysis
* Interactive charts and visual storytelling

## Tools Used

* Python
* Plotly
* Pandas
* NumPy
* Dash

## How to Run the Project

1. Install Python on your system.
2. Clone or download this repository.
3. Install the required libraries:

```bash
pip install -r requirements.txt
```

4. Run the application:

```bash
python dashboard.py
```

5. Open your browser and visit:

```text
http://127.0.0.1:8050/
```
